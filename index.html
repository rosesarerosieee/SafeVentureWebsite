System Instruction Manual

After registration the user will be redirected to the main activity of the app which is the “MapsFragment”.

The app will now ask the user the following permissions:
- Camera
- Location
- Storage
- Notification

These permissions are required to make sure the app works properly.
After granting the permissions the user can now Activate the “Capture Intruder

1.Homepage Overview:

Upon logging into the system, users will be directed to the homepage. The homepage features a map display showing the user's location and the locations of their connected friends. The navigation bar is located on the right side of the homepage.

2.Navigation Bar:

Users Bar:

Accessible from the navigation bar on the right. Allows users to view the list of connected friends. Users can add friends by entering their generated code. Unfriending friends is also possible through this section.

Intruder Catcher Tab:

  • Clicking on this tab grants access to security measures.
  • Requires granting permission to access location services or enabling GPS.
  • Upon granting access, the system will activate security measures.
  • If an incorrect passcode is entered on the user's device, the system will capture an image.
  • The captured image and the last known location of the device will be sent to a designated email address.

    • 3. Intruder Catcher Instructions:

  • Click the hamburger menu icon on the top left part of the app and go to the “CaptureIntruder” nav.
  • In CaptureIntruder Activity click the “Grant Button” for the app to ask for Device Admin Permission. The purpose of Device Admin is for the app to detect unsuccessful unlock attempts of the phone.
  • After granting the device admin, click the “Choose Email” button to select the email you will use for receiving  the captured images of the intruder

    • 4. Adding Friends:

    To add friends to your network, follow these steps: Access the Users Bar from the navigation bar. Enter the generated code of the friend you wish to add. Upon successful entry of the code, the friend will be added to your network.

    5. Removing Friends:

    To unfriend a user, follow these steps: Access the Users Bar from the navigation bar. Locate the friend you wish to unfriend. Select the option to unfriend the user.

    6. Viewing Friends' Locations:

    The map displayed on the homepage will show the locations of all connected friends. Users can view the map to see the real-time locations of their friends.

    7. Security and Privacy:

    Ensure that you only grant access to the Intruder Catcher feature to trusted individuals. Exercise caution when adding friends and sharing your generated code. Review and adjust privacy settings as needed to protect your personal information.

    8. Disclaimer:

    The system is designed to enhance user experience and provide security measures. However, no system is entirely foolproof, and users should exercise caution and common sense when using the features provided.

    Uninstalling the Application

    Users will not be able to uninstall the app when the device admin is granted, to uninstall the app the users will need to revoke the Device Admin permission from the CaptureIntruder page.

    - Go to the CaptureIntruder page, click the “Revoke Permission” button.

    - After that users can now uninstall the app.
